How to Organize and Analyze Research Data

This lesson is intended for students undertaking research projects, guiding them on formatting and analyzing research data using Excel and PASW (SPSS).

Preparation Requirements

  • Microsoft Excel
  • PASW (SPSS) Statistics software or free trial

Prerequisites

Users should be familiar with Excel functions, descriptive and inferential statistics, and research design basics.

Formatting Data for Analysis

Follow these steps to format your raw data in Excel:

  • Rename variables intuitively, ensuring each variable name is unique and starts with a letter.
  • Change categorical data to dichotomous format (e.g., change "female" to "F" and replace "Yes" with "1" and "No" with "0").
  • Separate combined data values into individual columns for easier analysis.

Why Dichotomous Variables?

Using 0/1 values allows for more flexibility in statistical analysis, making it easier to isolate specific characteristics for testing.

Creating the Formatted Data Sheet

Copy data from the "Raw_Dataset" sheet into a new sheet labeled "Formatted_Data". Organize data to simplify statistical testing.

Running Descriptive Statistics in Excel

To run basic descriptive statistics, ensure the Analysis ToolPak is enabled in Excel, then use Data Analysis > Descriptive Statistics:

  1. Select the range of data (e.g., columns with perc_enjoy, perc_usefulness, etc.).
  2. Output the summary statistics to a new sheet labeled "Descriptive_Stats".

Inferential Statistics with PASW

To perform inferential statistics in PASW (SPSS), open the Formatted_Data sheet and run tests based on your research questions:

  • Example: To test perc_usefulness across tutorial types, use the Kruskal-Wallis test under Nonparametric Tests > K Independent Samples.
